# Strata Documentation

Strata documentation is organized by reader.

- [Strata Users](strata-users/README.md) is for people building personal automation apps with Strata. It starts with the shape of the system and moves gradually toward implementation details.
- [Component Developers](component-developers/README.md) is for Go developers building reusable Strata components.

Each detailed page owns one topic. Overview pages link across topics, but the canonical explanation lives in the page named for that subject.
